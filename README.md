Weather Dashboard adalah aplikasi cuaca modern yang memberikan pengalaman pengguna yang informatif, interaktif, dan mudah digunakan. 
Aplikasi ini menampilkan data cuaca real-time dari OpenWeatherMap API dalam antarmuka yang responsif dan dilengkapi beragam fitur. 
Pengguna dapat memantau kondisi cuaca saat ini, melihat prakiraan hingga lima hari ke depan, mencari kota di seluruh dunia, menyimpan kota favorit, 
serta menyesuaikan tema dan satuan suhu sesuai preferensi.
</p>

## Key Features
**1. Current Weather Display**
* Temperature, humidity, wind speed
* Weather condition with icons
* Location and timestamp
* Real-time updates every 5 minutes

**2. 5-Day Forecast**
* Daily weather prediction
* Min/Max temperature
* Weather icons and descriptions

**3. Search Functionality**
* Search weather by city name
* Auto-complete suggestions
* Save favorite cities

**4. Interactive Features**
* Toggle between Celsius and Fahrenheit
* Dark/Light theme toggle
* Refresh button for manual updates
* Loading indicator
